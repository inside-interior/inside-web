# Inside Web

Website for Inside Interior (https://insideinterior.net/)

## Requirements

- `node 12.18.3`

## Installation

1. `brew install graphicsmagick`
2. `npm install`
3. `npm run build`

# Deployment

1. `npm run build`
2. Do sanity check locally to make sure everything is alright by using `npm run serve`
3. `npm run deploy`
